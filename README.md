# Sptfy

Spotify API client library.

## Installation

```elixir
def deps do
  [
    {:sptfy, "~> ..."}
  ]
end
```

## Usage

See [the documentation](https://hexdocs.pm/sptfy).

## Development

```shell
# Unit tests
$ mix test

# Integration tests
$ SPOTIFY_TOKEN="..." mix test ./integration_test
```
